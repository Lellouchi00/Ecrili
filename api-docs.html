Enums & Types

ENUM Property Types القيم المقبولة لحقل type
القيم
"apartment"   // شقة
"villa"        // فيلا
"studio"       // ستوديو
"house"        // منزل
"duplex"       // دوبلكس
"penthouse"    // بنتهاوس
"commercial"   // تجاري
ملاحظة الباك اند
// عدد المنشورات لكل نوع يُرجع
// مع كل نوع في استجابة الفلتر
// مثال:
{
  "type": "apartment",
  "count": 86
}
ENUM Amenities القيم المقبولة لحقل amenities[] — فقط ما أُضيف عند الإنشاء
Essential Amenities
"elevator"     // Elevator
"parking"      // Parking
"security"     // Security 24/7
"ac"           // Air Conditioning
Additional Amenities
"wifi"         // WiFi
"balcony"      // Balcony
"pool"         // Pool
"gym"          // Gym
"garden"       // Garden
"furnished"    // Furnished
"pets_allowed" // Pets Allowed
"storage"      // Storage
"wheelchair"   // Wheelchair Access
الفصل بين Essential وAdditional هو فقط للعرض في الـ UI — في الباك اند كلها حقل amenities[] واحد. كل قيمة Essential تُعرض مع عدد المنشورات التي تحتويها (count) في الفلتر.
ENUM Sort Categories قيم category وترتيب الباك اند المقابل لها
جدول التصنيفات
القيمةمنطق الترتيب في الباك اندملاحظة
all بدون ترتيب خاص الافتراضي
most_popular ORDER BY views DESC عدد المشاهدات
nearest ORDER BY distance ASC يتطلب lat و lng في الـ query
latest ORDER BY createdAt DESC الأحدث أولاً
best_price ORDER BY price ASC الأرخص أولاً
trending ORDER BY (views + favorites) DESC في آخر 7 أيام نشاط الأسبوع الحالي
new_construction WHERE isNewConstruction = true ORDER BY createdAt DESC فلتر ثابت + ترتيب

POST /listings/:listingId/rating إضافة تقييم بالنجوم على منشور
Request
// Headers
Authorization: Bearer <token>

// Body
{
  "userId": "user_ahmed123",  // required
  "stars": 5              // required — integer 1..5
}
⚠
شرط الصلاحية: نفس قاعدة التعليق — يجب أن يكون userId في سجل مستأجري هذا المنشور.

تقييم واحد لكل منشور: إذا قيّم المستخدم المنشور مسبقاً → 409 Conflict. لا يوجد تعديل للتقييم.

بعد كل تقييم جديد يجب إعادة حساب averageRating في الباك اند وتحديثه على المنشور مباشرة.
Response 201
{
  "listingId": "listing_abc123",
  "userId": "user_ahmed123",
  "stars": 5,
  "newAverageRating": 4.7,
  "totalRatings": 33
}

// Errors:
// 400 — stars خارج نطاق 1..5
// 403 — ليس مستأجراً لهذا المنشور
// 409 — سبق وأضاف تقييماً
